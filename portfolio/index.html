Instajam Documentation, Github

Instajam is a JavaScript wrapper for Instagram's API. In the days before Instagram's web profiles, I was working on building a way to present my own photos on the web. After amassing a veritable slew of JSONP calls in my code, I decided to standardize them into a utility, and Instajam came to be.

The utility has gone through several rewrites over the years, but the core premise has remained the same: the utility defines a global, Instajam, on the root. You initialize the utility by calling its “init” method, passing in your Instagram client ID. After that, you can make calls to the their API with a simple method call, passing in options and a callback function.

smallworld.js Documentation, Github

A utility for efficiently drawing map overviews with markers. At Findery, parts of our design required us to render, at times, dozens of Google maps on a page. A single piece of content, our “note”, contained two maps: neighborhood-level, and an “overview”. Sometimes a page would show a lengthy list of notes, and rendering these maps became a performance burden. smallworld.js came from those performance woes.

smallworld.js uses HTML Canvas, and draws a GeoJSON data source directly into it. Writing smallworld.js mandated that I learn a bit about map math, at first a painful experience and then one I came to enjoy. The utility projects coordinates using the Mercator projection, with map data sourced from Natural Earth.

crossfade.js Documentation, Github

Crossfades two images drawn into a Canvas as you scroll down a page. Originally written as an experiment in re-creating one of Medium's design elements. This project allowed me to get my hands dirty with drawing image data into Canvas elements and implementing efficient draw loops using requestAnimationFrame.

simple-clone Github

A Sublime Text package for quickly cloning a file from one tab to another. My first (and so far only) dive into Python. Sublime Text comes with support for cloning files, or creating a “New View into File”, but it doesn't work in a way that proves useful for workflow. simple-clone sets up an easy keyboard shortcut that will duplicate a file into a new tab to the right or below your current file. Useful for quickly working in two parts of a lengthy file at once.

Companimation Documentation, Github

An extension for Compass providing CSS Animation mixins and preset animations based on Animate.css. Before the official Animation module landed in Compass, I published an extension of my own. Companimation, in addition to providing mixins for defining animations and keyframes, provided Animate.css animations as Sass placeholders, allowing a developer to quickly pull in only the animations they needed without bloating their output.

Those Green Mountains Website, Github

A design + development collaboration highlighting the beauty and history of the Green Mountains in Vermont. Craig Winslow, Josh Franklin and I collaborated on this project, with them handling design and with me handling code. I implemented parallax in two dimensions using CSS Transforms on multiple layers. The site also leverages the keyboard for navigation between “slides”.

Farmers Markets In… Github

Simple app for searching for farmers markets in an area, using the USDA Farmers Markets API.
